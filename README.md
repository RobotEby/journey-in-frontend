# Frontend Journey: when I started in frontend

## A collection of early frontend projects, revisited and refactored with improved practices

- A collection of interactive web applications and utilities built with modern JavaScript, HTML, and CSS. Each project focuses on different aspects of web development, from API integration to form validation.
 Projects

## Field validator

- An input validation application that supports various data types, including CPF, email, UUID, URL, and hexadecimal colors. Created with the Validator.js library and an elegant dark theme.
Secure password generator

- A tool for generating strong, customizable passwords with options for length and character sets.

## ZIP Code Application

- A Brazilian ZIP code search tool that integrates with postal APIs to retrieve address information.

## Joke Generator

- A fun application that searches for random jokes from various categories using external joke APIs.

## Select Pet

- An interactive pet selection interface with filtering and display features.

## Superhero API

- A superhero database application that retrieves and displays information about comic book characters.
